# Walkability In College Towns vs. Non-College Towns

A statistical analysis project looking into whether college towns in the United States have significantly better walkability than in non-college towns.

## Description

This project looks at the relationship between college town status and walkability.
The main research question we sought to answer was, 
"Is the mean Walkability Index significantly higher in college towns than in non-college towns in the United States?" 

Using R and R Markdown, we cleaned and merged 2 datasets.
We visualized this data and used a 2-sample t-test as our statistical inference method.

Our analysis produced an extremely small p-value, indicating a statistically significant difference in mean walkability between college towns and non-college towns.
College towns were found to have higher average walkability scores.

## Note
The walkability dataset we used exceeds Github's 100mb max file size so it has been omitted from the repository.

The dataset can be found at: https://catalog.data.gov/dataset/walkability-index and should be placed in the Data/ folder
